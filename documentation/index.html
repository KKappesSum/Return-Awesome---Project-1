Admiral

Class that contains one player's data.

new Admiral(num: number, pName: string)
Parameters
num (number) The number of ships being placed in the game.
pName (string) the name of the player.
Properties
config (Config) : a Config object.
numships (number) : the number of Ships in Admiral.
board (Grid) : a Grid to store the Admiral's game map, with dimensions given in config.
fleet (Array<Ship>) : an array of Ships, initialized empty.
afloat (number) : the number of ships that are still afloat.
name (string) : the player's name.

Config

Class containing configuration parameters for the game's setup.

new Config()
Properties
MAX_NUM_OF_SHIPS (number) : the maximum number of ships allowed in the game.
BOARD_SIZE (number) : the length of each side of the square game board.
oceanTypes (Object) : object containing the possible states of a cell of the game board: WATER, SHIP, MISS, or HIT; and the color values assigned to each state.

Exec

Top-level class containing an Admiral for each player and methods to run the game.

new Exec(adm1Name: string, adm2Name: string, numShips: number)
Parameters
adm1Name (string) Admiral 1's nickname.
adm2Name (string) Admiral 2's nickname.
numShips (number) number of playable ships per Admiral.
Properties
m_shipCount (number) : the number of playable ships per Admiral.
m_playerTurn (number) : 1 for the first player, or 2 for the second player.
admir1 (Admiral) : the Admiral object for the first player.
admir2 (Admiral) : the Admiral object for the second player.
